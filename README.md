Tool to generate an initial typst layout of articles for the [Die Probezeit Zeit](https://dieprobezeit.ch) Newsletter.

## Workflow (may be out of sync with actual code)
### Configuration
- Edition number
- Release date
- List of the following for the previews
  - Article file name
- List of the following for the main body
  - Article file name
  - Whether to have a header
  - Whether to have a footer
  - Author Kürzel
### Generates
An initial typst file
